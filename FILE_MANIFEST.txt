## 📞 SUPPORT & TROUBLESHOOTING

### If something doesn't work:

**Page doesn't appear:**
- Restart: `streamlit run app.py --logger.level=debug`
- Check: Navigate to http://localhost:8501

**Image upload fails:**
- Check: user_images/ folder exists
- Check: File permissions
- Try: Refresh page

**AI not responding:**
- Check: Gemini API key valid
- Check: satellite_config.json configured
- Check: Network connection

**Location not showing:**
- Check: Folium library installed
- Check: Browser JavaScript enabled
- Try: Refresh page

**Database errors:**
- Check: MySQL running
- Check: Database connected
- Check: Tables exist
